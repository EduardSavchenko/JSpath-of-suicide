clear - Очистить терминал 

git init - инициализация локальной папки

git add . - добавить все изменения 

git commit -m "" - добавить коммит в "" после можно push

git log --oneline - вывод всех коммитов в 

однострочный вариант с их id коммита

git chekout id коммита - переход к версии коммита

git reset - удалить до определить коммита id 
(все которые до удаляются) (удалить комиты но файлы не меняют)

git reset id commit--hard  - удаляется до опреденного коммита
и возвращает файлы до нужного момента

git revert - отменить опреденный опреденный коммит id

git clone  - "клонировать" репозиторий для работы
на другом месте работы (масло малянное)

*** ветки  / branch ***
 git branch название ветки - аля admin ветка
 git chekout admin - переход на ветку admin
 git branch -a - вывод всех веток и на
 какой находишься ты
